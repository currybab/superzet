# Code of Conduct

This project follows a simple standard:

- be respectful
- assume good intent, but address impact directly
- keep technical disagreement focused on the work
- do not harass, demean, or exclude other contributors

Unacceptable behavior includes personal attacks, discriminatory language, intimidation, doxxing, and sustained disruption of discussion spaces.

If you need to report a problem, contact the repository maintainers privately through GitHub rather than escalating in public when possible.

Maintainers may edit, hide, lock, or remove content and participation that harms the project or its contributors.
